# Escape Room

G10 Computer Science Final Project  

## Requirements

1. Python 3+ (Tested on 3.12.3)
2. Required Libraries, install with

    ```shell
    cd <PROGRAM_LOCATION>
    pip install -r requirements.txt
    ```

## Run

To run the program, run the `run.py` Python File.  

```shell
cd <PROGRAM_LOCATION>
```

On Windows

```shell
python run.py
```

On Mac/Linux

```shell
python3 run.py
```
